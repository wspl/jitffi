# JITFFI

A faster, JIT-based Foreign Function Interface (FFI) for Node.js.

## Status

### Supported Platform
- Windows (x86_64)
- MacOS (x86_64)
- Linux (x86_64)

### Planned Support Platform
- Windows (ia32)
- MacOS (aarch64)
